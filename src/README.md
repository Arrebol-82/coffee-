# Source Directory

`src/` contains the active Vue application source.

| Directory | Purpose |
| --- | --- |
| `api/` | Frontend API facade used by views and stores |
| `assets/` | Static assets imported by source files |
| `components/` | Shared UI components |
| `composables/` | Reusable Composition API helpers |
| `constants/` | Shared constants and display maps |
| `layouts/` | Route layout components |
| `lib/` | Third-party client setup |
| `mocks/` | MSW handlers and mock data |
| `router/` | Vue Router config and route guards |
| `services/` | Supabase-backed service functions |
| `stores/` | Pinia stores |
| `types/` | Shared TypeScript types |
| `utils/` | Small reusable utilities |
| `views/` | Route-level pages |
